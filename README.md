# Kotlin Presentation

## How to present

1. Install dependencies

   ```sh
   npm install
   ```

2. Serve the presentation in watch mode

   ```sh
   npm start
   ```

3. Create a pdf

    ```sh
    npm run pdf-export
    ```

    ⚠️ `npm start` has to be running in order to export pdf ⚠️

## Shortcuts

* Fullscreen: `f`
* Overview: `ESC` or `o`
* PresenterView: `s`
